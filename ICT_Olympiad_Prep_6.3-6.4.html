Operating Systems & Office Applications

Windows/Linux basics, file management, Word, Excel, PowerPoint fundamentals

6.3.1

Windows / Linux Basics

An Operating System (OS) manages a computer's hardware and provides the platform on which all other software runs. Two major OS families dominate desktop computing today: Microsoft Windows and Linux.

Microsoft released the first version of Windows in 1985 as a graphical layer running on top of MS-DOS. Windows 95 (1995) was a landmark release, introducing the now-familiar Start menu and taskbar, and later versions (XP, 7, 10, 11) refined this GUI-based, mostly closed-source (“proprietary”) approach.

Linux traces its roots to the GNU Project, started by Richard Stallman in 1983 to build a free, Unix-like operating system. In 1991, Linus Torvalds released the Linux kernel — the core component that manages hardware — which combined with GNU tools to form complete operating systems. Because Linux is open-source, its source code can be freely viewed, modified, and redistributed, leading to many different distributions (“distros”) such as Ubuntu, Fedora, and Debian.

Users interact with an OS either through a Graphical User Interface (GUI) — icons, windows, menus, and a pointer — or a Command Line Interface (CLI), where users type text commands directly into a shell or terminal. Both Windows and Linux support multitasking, running many programs at once by rapidly scheduling CPU time between them.

Key facts & history
  • Windows: first released 1985 (Microsoft); Windows 95 popularised the Start menu
  • GNU Project (Richard Stallman, 1983) → Linux kernel (Linus Torvalds, 1991)
  • Linux is open-source; many distributions exist (Ubuntu, Fedora, Debian)
  • Kernel = core OS component that directly manages hardware
  • GUI (icons/windows/pointer) vs CLI (typed commands)
6.3.2

File Management

Computers organise data using files (individual units of stored data) and folders/directories (containers used to group related files). The series of folders leading to a specific file — e.g., C:\Users\Name\Documents\report.docx — is called its file path.

A file's type is usually indicated by its file extension, the short code after the final dot in its name (such as .docx, .xlsx, or .pdf). Behind the scenes, the operating system organises files on a disk using a file system: Windows commonly uses NTFS (introduced with Windows NT, succeeding the older FAT/FAT32 systems), while Linux commonly uses ext4.

Everyday file management tasks include copying (duplicating a file), cutting (moving a file, removing it from its original location), compressing (packing files into a smaller archive such as a .zip, useful for storage and sharing), and using shortcuts or symbolic links, which point to a file's location without duplicating its data. Deleted files on Windows are typically sent first to the Recycle Bin rather than being erased immediately. File permissions (especially important in Linux) control who is allowed to read, write, or execute a given file.

Key facts & history
  • File = unit of data; Folder/Directory = container grouping files
  • File extension (e.g. .docx) indicates type; file path shows its full location
  • Windows: NTFS (modern) succeeded FAT32; Linux: commonly ext4
  • Copy = duplicate; Cut = move; Compress = pack into a smaller archive (e.g. .zip)
  • Shortcuts/symbolic links point to a file without duplicating it
6.3.3

MS Word Fundamentals

Microsoft Word, one of the most widely used word processing applications, was first released by Microsoft in 1983 for Xenix and MS-DOS systems, years before the company's own Windows existed. It has since become the default tool for creating, formatting, and editing text documents.

Key productivity features include Spell Check (flags potentially misspelled words), Track Changes (records edits made by collaborators so they can be reviewed, accepted, or rejected), and Find & Replace (quickly locates and substitutes text). For longer documents, Word can auto-generate a Table of Contents from heading styles, and supports Headers and Footers (repeating text at the top/bottom of every page) along with Footnotes for reference notes.

Mail Merge combines a single template letter with a list of names or addresses to generate many personalised copies at once — useful for things like certificates or invitations. Word also includes AutoSave/AutoRecover, which periodically saves backup copies to prevent data loss if the program crashes. Since Word 2007, documents are saved by default using the .docx file format.

Key facts & history
  • Microsoft Word first released: 1983
  • Track Changes: review and accept/reject collaborator edits
  • Mail Merge: template + data list → personalised documents
  • Default modern format: .docx (since Word 2007)
  • AutoSave/AutoRecover: periodic backup to prevent data loss
6.3.4

MS Excel Fundamentals

Microsoft Excel, a spreadsheet application used for calculations, data analysis, and visualisation, was first released by Microsoft in 1985 for the Apple Macintosh, with a Windows version following in 1987. An Excel file is called a workbook, made up of one or more worksheets (sheets); each sheet is a grid where a cell is the intersection of a row and column.

Every Excel formula begins with an equals sign (=) and can call built-in functions like SUM, AVERAGE, COUNT, and VLOOKUP (which looks up a value in one column and returns a related value from another). Cell references can be relative (they shift when copied to other cells) or absolute, fixed using dollar signs like $A$1 so they don't change when copied.

Excel's Pivot Tables let users summarise and analyse large datasets by dragging fields into rows, columns, and value areas, while Conditional Formatting automatically changes a cell's appearance (such as its colour) based on its value. Built-in chart types, including the pie chart (ideal for showing proportions of a whole) and bar/line charts, help visualise data trends.

Key facts & history
  • Microsoft Excel first released: 1985 (Mac), 1987 (Windows)
  • Workbook = file containing one or more worksheets; Cell = row/column intersection
  • Formulas start with “=”; common functions: SUM, AVERAGE, VLOOKUP
  • Relative references shift when copied; Absolute references ($A$1) stay fixed
  • Pivot Tables summarise data; Conditional Formatting highlights values automatically
6.3.5

MS PowerPoint Fundamentals

PowerPoint, the presentation software, was originally developed by Forethought, Inc. and released in 1987; Microsoft acquired the company shortly after launch and has developed it ever since. A presentation is built from individual slides, each of which can contain text, images, charts, and other objects.

A Slide Master defines a consistent design template — fonts, colours, and layout — applied automatically across all slides, saving time and keeping a presentation visually consistent. Movement effects fall into two categories: a transition controls how one whole slide changes to the next, while an animation controls how an individual object (like a bullet point or image) moves within a single slide.

During a live talk, Presenter View shows the speaker their notes and the upcoming slide, while the audience sees only the current slide on the main screen. For organising a deck, Slide Sorter view displays thumbnails of every slide so they can be easily reordered. Since PowerPoint 2007, presentations are saved by default using the .pptx file format.

Key facts & history
  • PowerPoint: released 1987 by Forethought, Inc.; acquired by Microsoft the same year
  • Slide Master: consistent design/layout applied across all slides
  • Transition = effect between slides; Animation = effect on an object within a slide
  • Presenter View: speaker sees notes + next slide; audience sees only the current slide
  • Default modern format: .pptx (since PowerPoint 2007)

6.4.1

Binary, Decimal, Octal & Hexadecimal

A number system is defined by its base (radix) — the count of unique digit symbols it uses. The decimal system we use daily is base 10 (digits 0–9), almost certainly descended from counting on ten fingers and formalised through the Hindu-Arabic numeral system, developed by Indian mathematicians and later transmitted to Europe via Arabic scholars.

Digital computers, however, operate internally using binary (base 2), representing all data with just two symbols, 0 and 1, because electronic circuits can reliably distinguish only two states (off/on). The modern binary system was formally described by Gottfried Wilhelm Leibniz in 1679, centuries before electronic computers existed. Each binary digit is called a bit; a group of 4 bits is a nibble, and a group of 8 bits is a byte.

Because raw binary numbers are long and hard for humans to read, programmers often use octal (base 8) and especially hexadecimal (base 16, digits 0–9 then A–F) as compact shorthand — each hexadecimal digit conveniently represents exactly 4 binary bits. Hexadecimal shows up constantly in computing, from memory addresses to web colour codes like #FF5733.

Key facts & history
  • Decimal = base 10 (Hindu-Arabic numerals); Binary = base 2; Octal = base 8; Hexadecimal = base 16
  • Binary system formally described by Leibniz, 1679
  • Bit = 1 binary digit; Nibble = 4 bits; Byte = 8 bits
  • 1 hex digit = exactly 4 binary bits (compact shorthand for binary)
  • Hex used in memory addresses and web colour codes (e.g. #FF5733)
6.4.2

Logic Gates

Logic gates are the basic building blocks of digital circuits, each performing one simple Boolean operation on one or more binary inputs to produce a single binary output. The core gates are AND (output 1 only if all inputs are 1), OR (output 1 if at least one input is 1), and NOT (simply inverts a single input, also called an inverter).

Combining these creates derived gates: NAND (AND followed by NOT) and NOR (OR followed by NOT) are called universal gates because any other logic gate — and therefore any digital circuit — can be built using only NAND gates (or only NOR gates). XOR (Exclusive OR) outputs 1 only when its two inputs differ, while XNOR outputs 1 only when they match.

Physically, modern logic gates are built from transistors etched onto silicon chips. The theoretical foundation for designing these circuits came from Claude Shannon's 1937 master's thesis, which showed that Boolean algebra could be used to analyse and simplify electrical switching/relay circuits — a discovery often called the birth of digital circuit design. Combinations of gates build more complex components, such as a half adder (adds two bits, producing a sum and carry) or a flip-flop (a gate-based circuit that stores a single bit, forming the basis of computer memory).

Key facts & history
  • AND (all 1s → 1), OR (any 1 → 1), NOT (inverts) are the core gates
  • NAND and NOR are “universal gates” — any other gate can be built from just one of them
  • XOR: outputs 1 when inputs differ; XNOR: outputs 1 when inputs match
  • Claude Shannon's 1937 thesis: applied Boolean algebra to switching-circuit design
  • Half adder (adds 2 bits) and flip-flop (stores 1 bit) are built from gate combinations
6.4.3

Boolean Algebra

Boolean algebra, the mathematical system underlying all digital logic, was introduced by English mathematician George Boole in his 1854 book An Investigation of the Laws of Thought. It works with variables that can only take one of two values — 0 (False) or 1 (True) — combined using the three fundamental operations AND, OR, and NOT.

A set of algebraic laws lets engineers simplify Boolean expressions, which directly reduces the number of physical logic gates needed in a circuit (lowering cost, complexity, and power use). These include the identity laws (A·1 = A, A+0 = A), the idempotent law (A·A = A), the commutative law (A+B = B+A) and the distributive law (A·(B+C) = A·B + A·C). Perhaps the most famous are De Morgan's theorems, which state that the complement of (A AND B) equals (NOT A) OR (NOT B), and the complement of (A OR B) equals (NOT A) AND (NOT B).

For circuits with a manageable number of variables, designers often use a Karnaugh map (K-map) — a grid-based graphical method — to spot patterns and simplify expressions visually rather than through pure algebra. It was Claude Shannon's 1937 thesis that first connected Boole's 19th-century abstract algebra to the practical design of real electrical switching circuits, directly enabling the digital computers built decades later.

Key facts & history
  • Boolean algebra introduced by George Boole, 1854 (“An Investigation of the Laws of Thought”)
  • Core operations: AND, OR, NOT; values restricted to 0 (False) / 1 (True)
  • De Morgan's theorems: complement of (A·B) = A'+B'; complement of (A+B) = A'·B'
  • Karnaugh map (K-map): graphical tool for simplifying Boolean expressions
  • Shannon (1937) linked Boolean algebra to real switching-circuit design
